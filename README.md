# 🏦 Customer Churn Prediction

A Machine Learning web application built using **Python**, **Scikit-learn**, and **Streamlit** to predict whether a customer is likely to leave a bank.

## 🚀 Features

- Customer Churn Prediction
- Interactive Streamlit Dashboard
- Professional User Interface
- Real-time Prediction
- Machine Learning Pipeline
- Probability Score

## 📊 Input Features

- Credit Score
- Geography
- Gender
- Age
- Tenure
- Balance
- Number of Products
- Has Credit Card
- Active Member
- Estimated Salary

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## 📂 Project Structure

```text
Customer-Churn-Prediction/
│── app.py
│── model.pkl
│── requirements.txt
│── README.md
```

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🌐 Deployment

This application can be deployed using **Streamlit Community Cloud**.
